## 🧩 Key Features

### Responsive & Adaptive UI
Every template uses a fluid grid and flexible media queries. Images resize intelligently, text reflows gracefully, and navigation collapses into hamburger menus on smaller screens. Your portfolio looks intentional whether viewed on an ultrawide monitor or a smartphone held in one hand.

### Project Showcase Engine
Built-in support forFeatured Projects, with dedicated sections for:
- Project thumbnails with overlay captions
- Lightbox-style image galleries
- Embedded video support (YouTube, Vimeo)
- Custom call-to-action buttons

### SEO & Performance Optimized
- Lazy loading for images (native and JavaScript fallback)
- Minified CSS and JavaScript where applicable
- Structured data for Portfolio, Person, and CreativeWork schemas
- Fast initial load via critical CSS inlining

### Contact & Engagement Tools
- Pre-styled contact form (works with Blogger's native form widget)
- Social media link fields with SVG icons
- Newsletter signup integration placeholder
- WhatsApp and Telegram direct chat buttons (disable if not needed)

### Customization Without Code

- Font pairing suggestions (Google Fonts loaded locally for privacy)
- Layout toggles: sidebar left, sidebar right, full-width, or grid mosaic
- Pre-built page templates: About, Resume, Project Detail, Gallery, Testimonials

### 24/7 Community Support Philosophy
While this is a static repository, the accompanying documentation and issue tracker operate with a "help-first" approach. Expect responses within 24 hours during business days, with detailed explanations rather than copy-paste solutions.

## 🚀 Getting Started

1. **Browse the Templates** – Each folder in this repository represents a complete Blogger template. Look for the `theme.xml` file inside each folder.
2. **Preview Locally** – Open the `preview.html` file in your browser to see the template's structure before uploading.
3. **Upload to Blogger** – In your Blogger dashboard, go to Theme → Backup/Restore → Upload the `theme.xml` file. No coding required.
4. **Customize** – Use the Blogger Theme Designer or edit the `custom.css` file for advanced tweaks. All major customizations are documented in each template's `README.md`.
5. **Populate Your Content** – Add your projects, bio, and contact information. The templates auto-arrange content based on labels and pages.
